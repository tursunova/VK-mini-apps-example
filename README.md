## Getting Started

Install Node.js

### Installing

Clone the project using the command:

```
git clone https://github.com/tursunova/VK-mini-apps-example.git
```

Go to the project folder:

```
cd VK-mini-apps-example
```

Install npm packages:

```
npm install
```
Run the project:

```
npm start
```
